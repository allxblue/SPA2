# Vue.js SPA Boilerplate
Vue.js base SPA boilerplate

## Features
- Vue Router 3
- Vuex
- SCSS
- nested router and layout


## Build Setup
``` bash
# clone repo
git clone 

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build
```
